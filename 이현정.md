# 🚀zoeshodu
> 고양이 호두에 대한 상세페이지  
> 웹화면과 모바일 반응형 페이지 만들기
> 
## 🛠️사용언어
- HTML
- CSS

## 📂폴더 구조

```text
src/
 ├─ index.html
 ├─ css/
 │ ├─ font.css
 │ ├─ main.css
 │ └─ reset.css
 └─ images/
```
## 시작 전 노트
> 1만 시간에서 마크업 잘못해서 다 뜯어고치느라 시간낭비
> 시작 전 마크업 꼼꼼하게 진행 (한다고 했는데 반응형까지 고려하지 못함)
